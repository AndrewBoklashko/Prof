Запуск програми

 

       Для запуску програми Prof  потрібно скористатися файлом, що має іконку .

       Після запуску програми на екрані з’явиться заставка програми на якій розташовані пункти меню.

       У цьому меню є наступні пункти:

Ø     Побудова  - перехід до редактора Prof для здійснення побудов;

Ø     Налаштування – перехід до налаштувань роботи програми;

Ø     Про програму – перехід до вікна з інформацією про автора, версію програми та її призначення;

Ø     Допомога – перехід до довідки, де можна отримати будь-яку інформацію, що стосується програми;

Ø     Вихід – вихід з програми у середовище ОС Windows;

Середовище програми та головне меню

 

      Вибравши у меню, яке відображається на заставці пункт Побудова, потрапляємо у робоче середовище програми Prof. Робоче середовище складається із:

·        Заголовка робочого вікна програми, у якому зліва розташовується іконка програми Prof, самої назва програми та ім′я файла, який створюється нею. За замовчуванням програма надає файлові ім′я Сцена1;

·        Головного меню програми;

·        Горизонтальної панелі інструментів та інформаційних індикаторів;

·        Вертикальної панелі інструментів;

·        Робочої області, що має чорне тло на якому відображаються координатні осі та масштабна сітка.

        Головне меню програми складається з наступних пунктів:

·        Файл;

·        Намалювати;

·        Інструменти;

·        Допомога.

Кожен з цих пуктів меню складається з окремих підпунктів.

      Пункт Файл призначений для роботи з файлами і має у своєму складі наступні підпункти:

·        Нова сцена;

·        Відкрити;

·        Зберегти;

·        Зберегти як …;

·        Роздрукувати …;

·        Вихід.

       Підпункт Нова сцена призначений для створення нового 3D зображення.

       Підпункт Відкрити призначений для відкриття створеного раніше і збереженого 3D зображення. Після вибору цього пункту з′являється діалогове вікно для задання необхідних атрибутів для відкриття файла (папка, ім′я файла, тип файла).

       Підпункт Зберегти призначений для збереження створеного у середовищі програми Prof 3D зображення під поточним іменем. Якщо файл зберігається вперше за допомогою цього пункту, то він збережеться під іменем, що його програма Prof надає за замовчуванням, тобто під іменем Сцена1.

       Підпункт Зберегти як … призначений для збереження створеного у середовищі програми Prof 3D зображення під іменем, що вказується користувачем. У цьому випадку також з′являється діалогове вікно для задання необхідних атрибутів для збереження файла (папка, ім′я файла, тип файла).

       Підпункт Роздрукувати … призначений для отримання паперової копії  створеного у середовищі програми Prof 3D зображення. Після вибору цього пункту з′являється діалогове вікно для задання необхідних атрибутів для друку (  тип принтера, сторінки, копії, масштаб ).

       Підпункт Вихід призначений для завершення роботи програми і виходу в ОС Windows.

  Пункт Намалювати для створення зображень просторових тіл і має у своєму складі наступні підпункти:

·        Стандартні фігури;

·        Складні фігури.

       Підпункт Стандартні фігури призначений для створення стандартних просторових фігур , і в свою чергу він ще складається з наступних підпунктів:

·        Площина;

·        Сфера;

·        Паралелепіпед;

·        Піраміда;

·        Диск;

·        Циліндр;

·        Конус/Зрізаний конус.

 

       Підпункт Складні фігури призначений для створення складних просторових фігур, шляхом обертання або переміщення деякої твірної лінії або контуру навколо нерухомої осі. Цей підпункт ще складається з двох підпунктів, які і реалізовують ці два способи побудови складних просторових фігур:

·        Тіла обертання;

·        Тіла напряму.

Для здійснення побудов цими способами, спочатку задається твірна лінія або контур.

Пункт Інструменти надає у розпорядження користувачу найважливіші інструменти редагування та перегляду створених зображень і  має у своєму складі наступні підпункти:

·        Виділити;

·        Камера;

·        Повернути;

·        Масштабувати;

·        Перемістити;

·        Перемістити До;

·        Видалити;

·        Інвертувати нормалі;

·        Приховати;

·        Текстура.

       Підпункт Виділити призначений для виділення всього зображення, або окремих об′єктів складного зображення. Інструмент виділення дуже часто потрібний при редагуванні зображення просторового тіла. Наприклад, для накладання текстури, потрібно спочатку виділити все зображення, або окремі його об′єкти.

       Цей підпункт в свою чергу складається ще з таких підпунктів:

·        Все;

·        Об′єкти.

      Підпункт Камера призначений для перегляду 3D зображення з різних сторін і в свою чергу складається з наступних підпунктів:

·        Перемісти;

·        Кут огляду;

·        Вид.

А підпункт Вид цього підменю в свою чергу складається з наступних підпунктів:

·        Локальний;

·        Орбітальний;

·        Зверху;

·        Зпереду;

·        Збоку.

      Підпункт Повернути призначений для повертання створених на сцені об’єктів по осях X Y Z.

      Підпункт Масштабувати призначений для масштабування ( зміни розмірів ) створеного зображення.

      Підпункт Перемістити призначений для переміщення об’єктів з однієї точки 3D простору в іншу. Для цього необхідно затиснути праву клавішу миші,  перемістити 3D курсор у потрібну точку і відпустити клавішу.

      Підпункт Перемістити До призначений для переміщення об’єктів з однієї точки сцени в іншу . Для цього потрібно натиснути праву клавішу миші у тій точці 3D простору , де має бути центр переміщеного об’єкта.

      Підпункт Видалити призначений для видалення всього зображення або окремого виділеного його фрагмента.

       Підпункт Інвертувати нормалі призначений для акцентування відтінком кольору певної сторони об’єкта.

       Підпункт Приховати призначений для приховування всього зображення або окремих його об′єктів, а також для відміни режиму приховування. При виборі цього підпункту, з′являється додаткове підменю, що має наступний склад:

·        Приховати виділені;

·        Приховати Все;

·        Показати Все.

       Підпункт Текстура призначений для накладання різних текстур на зображення. Текстура — це картинка, графічне зображення, яке наноситься на сторони об'єктів. Завдяки текстурам, ми відрізняємо піщану доріжку від асфальтованої, дерев'яний ящик від кам'яної брили, цегельну стінку від скелі і т.д. У створеній програмі використовується 10 різних текстур.

       Пункт Допомога призначений для отримання довідкової інформації про програму та її використання і має у своєму складі наступні підпункти:

·        Довідка;

·        Про програму.

       Підпункт  Довідка  надає необхідну інформацію з використання програми Prof, а підпункт  Про програму надає інформацію про призначення програми, її версію та автора.

 

Панелі інструментів

       Під головним меню програми розташовується горизонтальна панель інструментів ( Мал. 10 ) , що складається з шести кнопок синього кольору. Ці кнопки забезпечують виконання найбільш вживаних дій при роботі з програмою Prof  і дублюють деякі підпункти пунктів головного меню.

       Розглянемо їх, кожну окремо ( зліва направо ):

Ø     « Нова сцена » – створення нової сцени( нового 3D зображення );

Ø     « Відкрити » – відкриття раніше збереженої сцени (3D зображення). При цьому з′являється додаткове діалогове вікно;

Ø     « Зберегти » – збереження поточної сцени у файл з розширенням .raw;

Ø     « Роздрукувати » – вивід зображення поточної сцени (3D зображення) на принтер;

Ø     « Довідка » – аналогія пункту « Допомога » у меню розташованому на  заставці програми;

Ø     « Вихід » – вихід з програми Prof.

      Правіше піктограм цих кнопок є три кнопки з написами X, Y, Z відповідно, а також зелене (за замовчуванням) поле для відображення поточного кольору побудов і ще правіше - напис «Перегляд». Якщо натиснути на будь-яку з кнопок X, Y, Z, то заблокується відповідна вісь і переміщення курсора по цій осі стане неможливим, а також недоступними стануть інші операції по цій осі. Під цими кнопками, у відповідних полях відображаються координати 3D курсора по відповідній осі.

       Зелене поле справа відображає поточний колір яким будуть будуватися об’ємні фігури. Натиснувши на це поле, з’являється діалог де ви можете змінити поточний колір.

       А правіше від цього поля розташоване поле для відображення дії, яка виконується в даний момент з графічними об’єктами.

       Зліва у вікні редактора є ще вісім кнопок оранжевого кольору. Їх призначення – побудова об’ємних фігур.

       Отже, перша зверху піктограма « Площина » - призначена для побудови площин у просторі. Натиснувши її, у змодельованому 3D просторі в центрі вікна, можна побудувати площину.

      Друга піктограма « Сфера » - призначена для побудови сфер..

     « Паралелепіпед » – третя піктограма призначена для побудови паралелепіпедів.

     « Піраміда » – піктограма для побудови чотирикутної піраміди..

     « Диск » – піктограма, призначена для побудови суцільних дисків і дисків з отвором посередині.

       « Циліндр » – за допомогою цієї піктограми можна будувати циліндр.

       « Конус/Зрізаний конус » – за допомогою цієї піктограми можна будувати конус або зрізаний конус.

       Трохи нижче від цієї групи піктограм для побудови стандартних просторових фігур, розташовуються  дві піктограми для побудови більш складних тіл . Це піктограми:

·        « Тіло обертання »;

·        « Тіло напряму ».

       « Тіло обертання » - піктограма для побудови поверхонь обертання другого порядку  шляхом обертання деякої твірної лінії або контуру навколо нерухомої осі.

      « Тіло напряму » - піктограма для побудови поверхонь другого порядку  шляхом переміщення деякої твірної лінії або контуру у вказаному напрямі.

 

 Використання програми

 

       Після запуску програми Prof  , на екрані з’явиться заставка програми на якій відображене початкове меню. Якщо робота з програмою відбувається вперше, то доцільно спочатку отримати більш детальну інформацію про програму, її можливості,  про основи роботи з нею. Для цього, потрібно послідовно скористатися пунктами цього меню «Про програму» та «Допомога ».

        Коли ж вже є певні навики роботи з програмою, то роботу можна розпочинати з пункту меню «Побудова», скориставшись перед цим пунктом «Налаштування», якщо у цьому виникає потреба.

       Після вибору пункту «Побудова», відбувається перехід у робоче середовище програми Prof. Основним елементом цього середовища є 3D простір, у якому і здійснюватимуться всі побудови і всі дії, пов′язані з їх переглядом та редагуванням. 3D простір програми - це чорний фон, на якому нанесені координатні осі та сітка. Координати в тривимірному просторі формують трійку (x, y, z). Координати x, y, z для тривимірної декартової системи можна розуміти як відстані від точки до відповідних площин: yz, xz, та xy. Тривимірна декартова система координат є дуже популярною, тому що відповідає звичним уявам про просторові виміри — висоту, ширину та довжину (тобто три виміри).

      Знизу вверх розташовується вісь ординат (Y), зліва направо розташовується вісь абсцис (X) і з екрану «на користувача» – проходить вісь аплікат (Z). Додатні напрями цих осей зображені червоним кольором, а від’ємні: вісь X – зеленим кольором, вісь Y фіолетовим  і синім  вісь Z. Також на площину XZ нанесена координатна сітка. Одна клітинка цієї сітки має розміри 1×1.

       І найважливішим елементом 3D простору є 3D курсор.3D курсор має вигляд білої стрілки. Для того, щоб перейти від звичайного курсора до 3D курсора і навпаки, потрібно натиснути клавішу з англійською літерою « M ». 3D курсор рухається наступним чином: при русі мишею ззаду вперед і зліва направо – 3D курсор рухатиметься з від’ємного у  додатній напрям осей X і Z відповідно. При русі коліщатком миші ззаду вперед - 3D курсор рухатиметься з від’ємного у додатному напрямі осі Y. Якщо ж коліщатко миші затиснути, то при русі миші ззаду вперед 3D курсор буде рухатися знизу вверх по осі ординат.

      При виборі інструменту « Площина » у змодельованому 3D просторі в центрі вікна, можна побудувати площину. Площина – алгебраїчна поверхня першого порядку, що задається у декартовій системі координат рівнянням першого степеня. В програмі Prof площина – це прямокутник, який завжди лежить в площині XZ, тому її можна визначити двома точками. Для побудови площини необхідно затиснути ліву клавішу миші для визначення першої точки і відпустити її в тому місці, де має бути точка діагонально протилежна першій. Після цього буде змодельована площина поточним кольором ( за замовчуванням він зелений  ). Але поточний колір завжди можна змінити на інший, клікнувши лівою клавішею миші по індикатору відображення кольору у верхній частині вікна. Після цього, у палітрі кольорів, що з′явиться, вибрати бажаний колір.

       Для побудови сфери, використовуємо піктограму «Сфера». Вона будується досить просто: затискається ліва клавіша миші у тому місці робочого простору, де має бути центр сфери і  графічний курсор рухається у певному напрямі, таким чином задаючи радіус сфери. Після задання радіуса, ліва клавіша миші відпускається і на екрані з′являється сфера.

       Піктограма « Паралелепіпед » – призначена для побудови паралелепіпедів. Паралелепіпед будується аналогічно як і площина - через задавання двох діагонально протилежних вершин.

        Для побудови піраміди використовується інструмент з піктограмою «Піраміда» Основою піраміди є площина. Тому для її побудови спочатку задається площина, а потім ще одна точка, яка є вершиною піраміди.

     « Диск » – піктограма, призначена для побудови дисків. У програмі Prof диск може бути побудований у двох видах:

·        Звичайний диск;

·        Диск з отвором в центрі.

      В першому випадку достатньо просто один раз натиснути ліву клавішу миші в точці, де має бути центр диска, а другий раз на відстані радіуса диска від його центра.

      В другому випадку потрібно затиснути ліву клавішу миші в центрі диска і відпустити на відстані радіуса отвору диска, потім ще раз натиснути на відстані радіуса самого диска. Така поверхня у математиці має також назву – тор.

       Для побудови циліндра використовується піктограма « Циліндр ». Щоб побудувати циліндр потрібно натиснути ліву клавішу миші там, де має бути центр його нижньої основи і відпустити на відстані радіуса нижньої основи. А потім вказати точку, яка буде на відстані висоти від центру нижньої основи.

      Для побудови конуса або зрізаного конуса використовується піктограма «Конус/Зрізаний конус ». Будуються ці фігури у програмі аналогічно циліндру. Тут має значення те, де вказуватиметься третя точка.

        Для побудови більш складних тіл використовуються інструменти з піктограмами:

·        « Тіло обертання »;

·        « Тіло напряму ».

       « Тіло обертання » - піктограма для побудови поверхонь обертання другого порядку ( наприклад: параболоїд,гіперболоїд, еліпсоїд, конічні поверхні і т.д. ) шляхом обертання деякої твірної лінії або контуру навколо нерухомої осі.

      « Тіло напряму » - піктограма для побудови поверхонь другого порядку  шляхом переміщення деякої твірної лінії або контуру у вказаному напрямі. Такі поверхні у математиці мають ще назву – кінематичні поверхні. Кінематичну поверхню ( тіло напряму ) визначають:

·        твірна;

·        напрямні елементи;

·        умови зміни розташування та форми твірної під час    руху, які задають її алгоритмічну частину.

     Сукупність положень твірної складає лінійчатий каркас поверхні. Така поверхня неперервна.

     Тіла напряму - це незакономірні поверхні, що називаються також топографічними.

     Створене програмою Prof зображення має досить реальний і зрозумілий вигляд, його можна видозмінювати, масштабувати, обертати для здійснення всебічного аналізу.

 

 

 Основні технічні вимоги та налаштування програми

 

        Мінімальними системними вимогами для використання програми Prof є:

Ø     Операційна система: Windows XP SP3, Windows Vista (всі версії), Windows 7 ;

Ø     Процесор з тактовою частотою 2 ГГц і більше ( для процесорів AMD вимагається підтримка SSE2 );

Ø     Память: 512 Мб і більше;

Ø     Відеокарта: сумісна з OpenGL 2.1 або DirectX 9.0c, з обсягом відеопам′яті 128 Мб і більше;

Ø     Дисплей з роздільною здатністю 1280x1024;

Ø    Звукова карта сумісна з DirectX®: 9.0c;

Ø      Вільний простір на жорсткому диску ( інтерфейс SAS або SATA швидкого доступу ) для установки програми ( 21 Мб ) і додатковий вільний дисковий простір для збереження результатів роботи;

Ø     Миша обов′язково;

Ø     Привід DVD-ROM;

Ø     Порт: 1 вільний USB-порт;

Ø     Принтер: кольоровий  принтер.

       Налаштування програми здійснюються за допомогою пункту меню Налаштування, яке відображається на заставці програми. У вікні налаштування можна:

Ø     збільшити або зменшити чутливість 3D курсора, тобто збільшити або зменшити час реакції курсора на рух мишки;

Ø     включити або виключити вертикальну синхронізацію, яка дозволяє позбутися мерехтіння зображення;

Ø     включити або виключити згладжування. Згладжування використовується для того, щоб робити межі кривих ліній більш гладкими, прибираючи «зубці», які виникають на краях об’єктів;

Ø     включити або виключити перехід до побудови. Включення цього режиму дозволяє оминути відображення заставки і переходити зразу до побудови зображення;

Ø     змінити частоту обновлення екрану. Її збільшення зменшує мерехтіння екрану, робить більш плавними повороти, масштабування.
